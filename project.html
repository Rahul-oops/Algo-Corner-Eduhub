Project 3: Snake Ladder Game
Snake and Ladder is a popular board game enjoyed by many. The game follows simple rules:
1.The objective is to be the first player to reach 100 points.
2.Each player takes turns to roll a dice and move forward accordingly.
3.Landing on a snake reduces the player's points, while landing on a ladder increases them.
4.The game continues until one player reaches or exceeds 100 points.

We can implement these rules into a C program to create a Snake and Ladder application. The program's functionality will include:

1.Two players participating in a single game.
2.Generating random dice rolls to determine movement.
3.Tracking each player's points using variables.
4.Ending the game once any player reaches or surpasses 100 points
