Computer Fundamentals

Question 1: What is a computer?

A computer is an electronic device that processes data to perform various tasks.

Question 2: What are the main parts of a computer?

The main parts of a computer include the central processing unit (CPU), memory (RAM), storage devices (hard drive, SSD), input devices (keyboard, mouse), and output devices (monitor, printer).

Question 3: What is the CPU?

The CPU, or central processing unit, is the "brain" of the computer responsible for executing instructions and performing calculations.

Question 4: What is RAM?

RAM, or random access memory, is temporary memory used by the computer to store data and programs that are currently in use.

Question 5: What is a hard drive?

A hard drive is a storage device used to store data permanently on a computer.

Question 6: What is an operating system?

An operating system is software that manages computer hardware and provides common services for computer programs.

Question 7: What is a file?

A file is a collection of data stored on a computer under a specific name and location.

Question 8: What is a folder?

A folder is a container used to organize and store files on a computer.

Question 9: What is a mouse used for?

A mouse is an input device used to move the cursor on the screen and interact with graphical user interfaces.

Question 10: What is a keyboard used for?

A keyboard is an input device used to enter text, numbers, and commands into a computer.

Question 11: What is a monitor used for?

A monitor is an output device used to display information and graphics generated by the computer.

Question 12: What is a printer used for?

A printer is an output device used to produce physical copies of documents and images on paper.

Question 13: What is a web browser?

A web browser is software used to access and view websites on the internet.

Question 14: What is a search engine?

A search engine is a website or software that allows users to search for information on the internet.

Question 15: What is a virus?

A virus is a type of malicious software designed to replicate itself and infect computer systems, often causing harm to files or stealing information.

Question 16: What is a firewall?

A firewall is a security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.

Question 17: What is a password?

A password is a secret combination of characters used to authenticate a user and gain access to a computer, system, or service.

Question 18: What is a URL?

A URL, or Uniform Resource Locator, is a web address used to specify the location of a resource on the internet.

Question 19: What is an email?

An email is an electronic message sent over a computer network from one person to another.

Question 20: What is a browser cache?

A browser cache is a temporary storage location on a computer where web pages and files are stored to improve website loading times.

Question 21: What is a cursor?

A cursor is a graphical representation (usually an arrow or blinking line) on the computer screen that indicates where the next action will occur, such as typing or clicking.

Question 22: What is a desktop?

A desktop is the primary graphical interface of a computer operating system, where users can access files, folders, and applications.

Question 23: What is a recycle bin?

A recycle bin is a feature in the Windows operating system where deleted files are temporarily stored before they are permanently removed from the system.

Question 24: What is a right-click?

Right-clicking refers to the action of pressing the right button on a mouse to display a context menu with options relevant to the current task or selection.

Question 25: What is a hyperlink?

A hyperlink is a clickable element on a web page or document that, when clicked, directs the user to another location, such as another web page or a specific section within the same document.

Question 26: What is a screenshot?

A screenshot is a digital image captured by the computer to record the visible items displayed on the screen at a specific moment.

Question 27: What is a webcam?

A webcam is a camera device connected to a computer that captures video and images for video conferencing, live streaming, or other purposes.

Question 28: What is a USB port?

A USB port is a standard interface on computers and other electronic devices used to connect peripherals such as keyboards, mice, printers, and external storage devices.

Question 29: What is a software update?

A software update is a release of patches, bug fixes, or new features for software programs to improve performance, security, or functionality.

Question 30: What is a taskbar?

A taskbar is a horizontal bar displayed on the screen's edge containing buttons or icons representing open programs, system notifications, and shortcuts to frequently used applications.
